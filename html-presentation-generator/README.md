# HTML Presentation Generator

Generate professional HTML presentations from structured notes using Claude Code.

## What This Skill Does

Transforms your structured notes into a complete, self-contained HTML presentation with:
- Keyboard navigation (arrow keys)
- Chapter sidebar with click-to-jump
- Slide counter
- Smooth transitions
- Dark or light theme options

## When to Use

- Workshop presentations
- Product demos
- Conference talks
- Pitch decks
- Training materials
- Any presentation where you want more control than PowerPoint allows

## Quick Start

```
Create a presentation from these notes:

# My Topic
## Section 1
- Key point 1
- Key statistic: 73%
...
```

## Key Features

| Feature | Description |
|---------|-------------|
| Single file | All CSS/JS embedded, no dependencies |
| Keyboard nav | Left/right arrows, spacebar, home/end |
| Chapter menu | Left sidebar, click to jump |
| Themes | Dark (recording) or light (traditional) |
| Responsive | Works on any screen size |
| Portable | Deploy anywhere, share easily |
